Built on the frontier

WebGPU-first, with a WebGL2 safety net.

Compute shaders, storage buffers, and modern pipelines — behind an engine-agnostic renderer interface, so a scene authored once runs on any backend. The very page you're on is proof: it just detected your device and picked the best path.

GPU-first

If it can run as a compute pass, it does. That's how you get real-time fluids and millions of particles in a tab.
